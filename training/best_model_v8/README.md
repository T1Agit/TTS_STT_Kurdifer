# VITS v8 Fine-tuned Model

This directory should contain the fine-tuned VITS TTS model for Kurdish (Kurmanji).

## Expected Files

Place your trained model files here (generated by `train_vits.py`):

```
training/best_model_v8/
├── config.json              # Model configuration
├── pytorch_model.bin        # Model weights
├── tokenizer_config.json    # Tokenizer configuration  
├── vocab.json              # Vocabulary mapping
└── special_tokens_map.json # Special tokens
```

## Training Information

- **Base Model**: `facebook/mms-tts-kmr-script_latin`
- **Architecture**: VITS (36M parameters)
- **Training Details**:
  - Best loss: 0.6326 (epoch 9 of 14)
  - Sample rate: 16000 Hz
  - Fine-tuned components: text_encoder, duration_predictor
  - Frozen components: decoder
  
## How to Generate This Model

1. Prepare your Kurdish dataset:
   ```bash
   python prepare_data.py --output_dir training --max_samples 5000
   ```

2. Train the model:
   ```bash
   python train_vits.py --epochs 10 --output_dir training/best_model_v8
   ```

3. The trained model files will be automatically saved here.

## Using the Model

Once the model files are in place, the TTS service will automatically detect and use them:

```python
from tts_stt_service_base44 import TTSSTTServiceBase44

service = TTSSTTServiceBase44()
result = service.text_to_speech_base44(
    text="Silav, tu çawa yî?",
    language="kurdish",
    model_version="v8"  # Use the trained model
)
```

## Model Comparison

You can compare the original Facebook MMS model with your trained v8 model:

```bash

curl -X POST http://localhost:5000/tts \
  -H "Content-Type: application/json" \
  -d '{"text": "Silav", "language": "kurdish", "model": "original"}'

  -H "Content-Type: application/json" \

```
